--------------------------
Setting up an EC2 Instance
--------------------------

Step 1: Login into your AWS Console
Step 2: Create an EC2 Instance
	1. Give a name to the EC2 Instance
	2. Select the AMI for the EC2 Instance
	3. Create a new Login pair for that EC2 instance
	4. While creating, select .pem for Linux-based hosts and .ppk for windows-based hosts
	5. Configure the Network, enable ssh, http and https traffic
	6. Create the instance
Step 3: Open a Terminal in Linux Host and navigate to the directory where the login pair is saved
Step 4: Configure the file permissions by running the command
	chmod 400 keypair.pem #for .pem files
	chmod 400 keypair.ppk #for .ppk files
Step 5:	Copy the public IP of the created EC2 instance
Step 6: Note down the userName of the EC2 Instance:
	1. Amazon Linux -> ec2-user
	2. Ubuntu -> ubuntu
	3. Debian -> admin/debian
	4. RHEL -> ec2-user
Step 5: Connect the EC2 Instance with ssh by using the command:
	ssh -i keypair.pem username@public_ip
Step 6: Ensure that the ssh is logged in sucessfully

-------------------------- 
Install Python Environment
-------------------------- 

Step 1: Update apt packages by using the following commands:
	sudo apt update
	sudo apt upgrade
Step 2: Install Python
	sudo apt install python3 python3-pip python3-venv
Step 3: Verify the Python installation
	python3 --version
Step 4: Create a virtual Environment
	python3 -m venv venv
Step 5: Activate the virtual Environment
	source venv/bin/activate
Step 6: Install Flask packages
	pip3 install flask

----------------
Create Webserver
----------------

Step 1: Create a python file
	touch app.py
Step 2: Edit the python file using nano editor
	nano app.py
Step 3: Type in the following python code
	from flask import Flask

	app = Flask(__name__)

	@app.route("/")
	def home():
    	return "Hello, World!"

	if __name__ == "__main__":
    	app.run(host="0.0.0.0", port=5000)
Step 4: Save the python file and exit the nano editor
Step 5: Run the python file
	python3 app.pyp

-------------------
Output Verification
-------------------

Step 1: Open a new terminal window
Step 2: Connect the EC2 Instance with ssh by using the command:
	ssh -i keypair.pem username@public_ip
Step 3: Send in a curl request to the server by the commands
	curl EC2_IP:5000
Step 4: If there exists any output, then the server works fine
